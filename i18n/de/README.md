- Dies ist ein Plugin für [LifeOS](https://lifeos.vip), das bei der Anwendung der PARA-Methode mit periodischen Notizen und [usememos](https://www.usememos.com/).
- Es wird empfohlen, das [LifeOS-example](https://github.com/quanru/obsidian-example-lifeos/releases/latest/download/LifeOS_DE.zip) um es auszuprobieren.
- Weitere Tutorials finden Sie auf der [LifeOS für Obsidian Offizielle Website](https://lifeos.vip/de/)

![](https://obsidian-life-os.pages.dev/plugin/periodic-para-plugin-en.png)
