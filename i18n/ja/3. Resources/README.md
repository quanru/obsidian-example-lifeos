- リソースとは、継続的に関心を持つトピックやテーマのことである。

リソースとは、外部から取得した情報や知識であり、エリアをある程度支え、深化させるものを指す。

例：

- **興味・関心**
  - Webデザイン
  - 木工
  - ディスクゴルフ
  - バイオハッキング
- **トピック**
  - 心理学
  - 政治学
  - リーダーシップ
  - 誠実さ
- **アセット**
  - ストックフォト
  - タイポグラフィの参考資料
  - マーケティングのスワイプファイル
  - おすすめ製品
  - コードスニペット

これらのリソースから継続的に栄養を吸収することで、プロジェクトが成果を上げるための土台となり、エリアが継続的に深化していく。

そのため、リソースはP.A.R.Aシステムの欠かせない要素でもある。
