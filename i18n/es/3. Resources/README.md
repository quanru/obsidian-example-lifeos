- Los recursos son un tema o materia de interés continuo.

Los recursos se refieren a la información y el conocimiento que adquieres de fuentes externas, los cuales respaldan y profundizan tus áreas en cierta medida.

Por ejemplo:

- **Intereses**
  - Diseño web
  - Carpintería
  - Disc golf
  - Biohacking
- **Temas**
  - Psicología
  - Política
  - Liderazgo
  - Integridad
- **Recursos digitales**
  - Fotos de stock
  - Referencias tipográficas
  - Archivos de inspiración de marketing
  - Recomendaciones de productos
  - Fragmentos de código

Nutrirse continuamente de estos recursos proporciona la base para que los proyectos tengan impacto y para que las áreas se profundicen de manera constante.

Por lo tanto, los recursos también son una parte esencial del sistema P.A.R.A.
