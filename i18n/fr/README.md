- Ceci est un plugin pour [LifeOS](https://lifeos.vip), qui aide à pratiquer la méthode PARA avec des notes périodiques et [usememos](https://www.usememos.com/).
- Il est recommandé de télécharger [LifeOS-example](https://github.com/quanru/obsidian-example-lifeos/releases/latest/download/LifeOS_FR.zip) pour l'essayer.
- Pour plus de tutoriels, rendez-vous sur [Site Officiel LifeOS pour Obsidian](https://lifeos.vip/fr/)

## Remerciements

J'ai également utilisé le code de ces excellents plugins :

- [PARA](https://fortelabs.com/blog/para/). Merci à Tiago Forte d'avoir inventé une méthode d'organisation aussi remarquable.
- [PARA Starter Kit](https://forum.obsidian.md/t/para-starter-kit/223). Merci à cotemaxime d'avoir créé un coffre de démarrage aussi excellent.
- [obsidian-periodic-notes](https://github.com/liamcain/obsidian-periodic-notes). M'a donné l'inspiration pour développer ce plugin.
- [obsidian-dataview](https://github.com/blacksmithgu/obsidian-dataview). Le plugin sous-jacent utilise l'interface de requête de dataview.
- [Templater](https://github.com/SilentVoid13/Templater). Offre une création de modèles découplée, permettant à un très grand nombre de plugins de créer des fichiers grâce à lui.
