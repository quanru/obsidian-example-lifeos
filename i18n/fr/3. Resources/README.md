- Les ressources sont un sujet ou un thème d'intérêt continu.

Les ressources désignent les informations et connaissances acquises auprès de sources externes qui soutiennent et approfondissent vos domaines dans une certaine mesure.

Par exemple :

- **Intérêts**
  - Design web
  - Menuiserie
  - Disc golf
  - Biohacking
- **Sujets**
  - Psychologie
  - Politique
  - Leadership
  - Intégrité
- **Actifs**
  - Photos de stock
  - Références typographiques
  - Fichiers d'inspiration marketing
  - Recommandations de produits
  - Extraits de code

Puiser continuellement dans ces ressources fournit une base permettant aux projets d'avoir un impact et aux domaines de s'approfondir en permanence.

C'est pourquoi les ressources constituent également une partie essentielle du système P.A.R.A.
