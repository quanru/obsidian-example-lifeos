- Recursos são um tópico ou tema de interesse contínuo.

Recursos referem-se às informações e conhecimentos adquiridos de fontes externas que apoiam e aprofundam suas áreas até certo ponto.

Por exemplo:

- **Interesses**
  - Design web
  - Marcenaria
  - Disc golf
  - Biohacking
- **Tópicos**
  - Psicologia
  - Política
  - Liderança
  - Integridade
- **Ativos**
  - Fotos de banco de imagens
  - Referências tipográficas
  - Arquivos de referência de marketing
  - Recomendações de produtos
  - Trechos de código

Extrair continuamente conhecimento desses recursos fornece a base para que os projetos gerem impacto e para que as áreas se aprofundem progressivamente.

Portanto, os recursos também são uma parte essencial do sistema P.A.R.A.
