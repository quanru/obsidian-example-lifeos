- Recomenda-se baixar o [LifeOS-example](https://github.com/quanru/obsidian-example-lifeos/releases/latest/download/LifeOS_PT.zip) para experimentar.
- Para mais tutoriais, acesse [Site Oficial do LifeOS para Obsidian](https://lifeos.vip/pt/)
